# KiloEssentials
An essentials mod for fabric developed for the KiloCraft server

**IP:** 50kilo.org

## Setup

1. Edit build.gradle and mod.json to suit your needs.
2. Run the following command:
```
./gradlew build
```
3.Grab the generated Jar from `/build/libs/`
